# Lessons Learned

今回の最大の障害は Kubernetes ではなく Proxmox Firewall による VXLAN遮断だった。

RKE2 on Proxmox で Pod 通信障害が発生した場合は、

1. UDP 8472
2. flannel.1 RXカウンタ
3. Proxmox Firewall

を最優先で確認する。
